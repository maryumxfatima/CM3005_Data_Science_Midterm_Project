# Data Science Midterm: Automotive Price Prediction

## Overview
This project was completed as part of the **CM3005 Data Science course midterm**. The goal was to explore the use of **linear regression** in predicting automobile prices based on various features. The automotive industry was chosen due to its complex pricing dynamics and diverse variables that influence the final price of a vehicle.  

By applying data science techniques including **data preprocessing, exploratory data analysis (EDA), and linear regression modeling**, this project aims to uncover meaningful patterns in car pricing and provide insights for both consumers and industry stakeholders.

## Domain
**Automotive Industry** – The dataset includes detailed information about cars such as manufacturer, model, year, engine specifications, transmission, vehicle style, fuel type, and other characteristics. This allows for a comprehensive analysis of factors influencing automobile prices.

## Objectives
- **Unravel Pricing Patterns:** Identify the most important features that influence car prices and discover hidden correlations.  
- **Support Consumer Decisions:** Provide insights to help potential buyers make informed, data-driven choices.  
- **Industry Guidance:** Offer manufacturers and dealers a better understanding of how vehicle attributes affect pricing and perceived value.

## Dataset
The dataset was sourced from [Kaggle](https://www.kaggle.com/datasets/aminesnoussi/carpriceassignmentcsv/data), containing **over 10,000 entries** with both numerical and categorical features:  
- Make, Model, Year  
- Engine Fuel Type, Engine HP, Engine Cylinders  
- Transmission Type, Driven Wheels, Number of Doors  
- Market Category, Vehicle Size, Vehicle Style  
- Highway MPG, City MPG, Popularity, MSRP  

The combination of numerical and categorical features allows for robust modeling using linear regression.

## Methodology
The workflow for this project included:  
1. **Data Preprocessing:** Handling missing values, removing duplicates, and cleaning the dataset.  
2. **Exploratory Data Analysis (EDA):** Using visualizations and statistical summaries to understand feature distributions, trends, and relationships with the target variable (MSRP).  
3. **Feature Selection:** Identifying features strongly correlated with MSRP to improve model accuracy.  
4. **Linear Regression Modeling:** Applying a predictive model to estimate car prices and evaluate its performance.

## Key Insights
- Engine horsepower, number of cylinders, and vehicle year showed strong positive correlations with price.  
- Transmission type, vehicle style, and make significantly influence MSRP.  
- Certain features such as popularity and fuel type had a lesser impact but still contributed to model performance.  

Visualizations like **scatter plots, box plots, bar charts, pairplots, and correlation heatmaps** were crucial in identifying these patterns.

## Skills Demonstrated
- Data preprocessing and cleaning  
- Exploratory data analysis (EDA) and data visualization  
- Understanding and applying linear regression  
- Correlation analysis and feature selection  
- Insight generation from real-world datasets  

## Conclusion
This project highlights the power of **linear regression in predicting automotive prices** and demonstrates the process of turning raw data into actionable insights. While this was a midterm exercise, it serves as a practical example of applying data science techniques to real-world problems.
